DeepSeek R1 MIT License
MIT 开源
DeepSeek R1 采用 MIT License，允许蒸馏和商用。开源引爆全球关注 → 极低成本获得媒体曝光 → 建立技术权威性
OpenAI
公司转型历史
完全闭源
从非营利开源转为商业闭源。最强能力 → API 定价权 → 订阅收入。开源 = 竞争对手可复制，与商业模式根本冲突
PM 视角洞见
🏆
开源的好处
开发者生态、技术影响力、全球研究社区贡献、无需 API 管控
💰
闭源的好处
模型能力护城河、API 定价权、防止对手直接复制最新技术
🧭
趋势
越来越多厂商采用"小模型开源、大模型闭源"的双轨策略
🎯
选型建议
优先考虑开源程度高的模型做基础，节省成本；需要最强能力时再接入闭源 API
♟️ 开源是策略，不是慈善。
读懂厂商的开源逻辑，才能做出最适合自己产品的技术选型。不要被"开源"的标签迷惑。
